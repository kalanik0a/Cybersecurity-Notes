# Key Wrapping Ciphers
- Used specifically to encrypt other keys securely

### Examples
- **AES Key Wrap** (RFC 3394, RFC 5649)
    - **Usage**: Secure key exchange, key escrow
    - **Notes**: Ensures integrity and confidentiality of key material
