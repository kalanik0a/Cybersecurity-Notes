# DFIR Unfurl
Unfurl is a resource for deformulating various data structures like URLs, IDs, Encodings, and Timestamps

https://dfir.blog/unfurl/

# Subjects Covered
- General URLs
- Search Engine URLs
    - Google Search URLs
    - Bing Search URLs
    - DuckDuckGo Search URLs
- Social Media & Video Sites
    - Tweet on Twitter
    - Message on Discord
    - Video on TikTok
    - Video on YouTube
    - Search on Facebook
- IDs
    - UUIDv1
    - UUIDv4
    - Sonyflake
    - TikTok ID
    - KSUID
    - ULID
- Encodings
    - Protobuf in Hex
    - Base64-encoded Protobuf
    - Base64
    - Base64-encoded Zip
- Timestamps
    - Epoch Seconds
    - Epoch Second in Hex
    - Epoch Milliseconds
    - WebKit
    - Windows FileTime
    - DateTime Ticks
    - Mac Absolute Time
